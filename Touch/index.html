Touch

Touch is fully-dynamic keyboard. We build hardware right in monitor, so the keyboard is projecting right in table. Keys and buttons appear and disappear in right time. With our platform everybody can make own keyboard for any software.

First WorldWide Keyboard

With no hardware on table, we have bigger place for working. The keyboard appear in right time and dissapear, when you do not need it.

The end of shortcuts

Now, everytime you press ctrl, alt or shift, keyboards changes classic alphabet keys to symbols or events like copy or paste.

Dynamic Change

With touch, using your device will be more powerful. Keys and buttons appear and disappear in the right time. Control space will be bigger and more powerful.
